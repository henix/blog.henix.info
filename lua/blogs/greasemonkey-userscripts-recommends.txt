<h3>什么是油猴脚本，我为什么要用它？</h3>
<p>　　扩展浏览器功能的方式，有这么几种：</p>
<ul>
	<li>各个浏览器的扩展（Extensions）或称插件（Plugins/Add-ons）：功能很强大，浏览器间不能通用</li>
	<li>油猴脚本，或称 userscripts：就是一段 javascript ，当访问指定的网站时由浏览器加载，浏览器间可以通用</li>
	<li>bookmarklets：一段 javascript 直接保存在书签栏里，要用时手动点</li>
</ul>
<p>　　可见油猴脚本具有很高的通用性，并且适用于那些需要跟目标网站一起加载的任务。</p>
<p>　　如果一个任务可以用 JavaScript 搞定，就可以写成油猴脚本或者 bookmarklets 而不需要浏览器插件。</p>

<h3>如何安装油猴脚本？</h3>
<ul>
<li>chrome 用户直接点。实际上，当打开一个以 .user.js 结尾的链接的时候就会提示安装，chrome 是把它当扩展安装了。</li>
<li>firefox 用户需要安装扩展 <a href="https://addons.mozilla.org/zh-CN/firefox/addon/greasemonkey/">GreaseMonkey</a> 。</li>
<li>opera 用户打开 Settings -&gt; Advanced -&gt; Content -&gt; JavaScript Options -&gt; User JavaScript Folder 然后选择一个文件夹并把 userjs 放在那个文件夹下面。</li>
</ul>
