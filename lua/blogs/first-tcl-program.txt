<p>　　最近学了下 <a href="http://www.tcl.tk/">Tcl/Tk</a> ，语法还比较容易理解，但很纠结：</p>
<pre class="brush: tcl;"># 递归版
proc fac {n} {
	if {$n == 1} {
		return 1
	} else {
		return [expr {$n * [fac [expr {$n-1}]]}]
	}
}

# 非递归版
proc fac2 {n} {
	set r 1
	for {set i 2} {$i &lt;= $n} {incr i} {
		set r [expr {$r * $i}]
	}
	return $r
}

puts [fac2 1000] ; # 高精度
</pre>
<p>　　全是方括号和花括号，而且还不能写错了……不过 Tcl 有个好处是整数运算是高精度的，速度也很快。</p>
